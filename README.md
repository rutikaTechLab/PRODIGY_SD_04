# 📖 About

**Sudoku Solver** is a modern desktop application that automatically solves **9×9 Sudoku puzzles** using the **Backtracking Algorithm**. Simply enter the puzzle, click **Solve**, and let the algorithm find the correct solution in seconds.

- 🧩 Solve any valid Sudoku puzzle

- 🎲 Load Sample Puzzle

# 🎮 How to Use

- ✍️ Enter the Sudoku numbers.
- ⬜ Leave empty cells blank.
- ▶️ Click **Solve Sudoku**.
- 🎉 View the completed Sudoku grid.
- 🧹 Use **Clear Board** to reset.
- 🎲 Click **Load Sample** to test with a predefined puzzle.

# 🧠 Algorithm

The application uses the **Backtracking Algorithm**, which:

- 🔍 Finds an empty cell
- 🔢 Tries numbers **1–9**
- ✅ Checks row, column, and 3×3 box validity
- 🔁 Backtracks when necessary
- 🎯 Continues until the puzzle is solved
